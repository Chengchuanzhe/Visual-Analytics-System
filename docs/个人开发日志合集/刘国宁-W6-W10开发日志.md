#### 2. 新颖图表需求分析

**分析过程**

第一轮: 痛点识别
- 现有图表: 散点图(2维) + 饼图(1维) + 雷达图(多维-径向)
- 痛点: 缺少时间维度 + 多维关联难以表示 + 大数据下维度探索困难
- 机会: 需要一种图表支持 8维以上 + 维度筛选 + 时间序列分析

第二轮: 方案评估
| 图表类型 | 维度支持 | 时间展示 | 大数据适配 | 选题匹配度 |
|---------|---------|--------|----------|----------|
| 平行坐标图 | 8+ | ✓优 | ✓优 | ✓优 |
| 热力图 | 3维 | ✓中 | ✓优 | ✗低 |
| 网络图 | 多维关联 | ✗差 | ✗差 | ✗低 |
| 树形图 | 分层关系 | ✗差 | ✓优 | ✗低 |

**决策: 平行坐标图**

理由：
1. 维度支持最多(8维以上)
2. 时间序列可通过颜色编码展示
3. 大数据量支持(可使用采样)
4. 符合时空多维分析需求
5. 用户可交互筛选维度

#### 2. 坐标轴映射实现

**核心计算逻辑**
```javascript
// 维度定义
const dimensions = [
  {key: 'sst', label: '海表温度(°C)', min: -2, max: 32},
  {key: 'salinity', label: '盐度(PSU)', min: 30, max: 38},
  {key: 'wind_speed', label: '风速(km/h)', min: 0, max: 20},
  {key: 'pressure', label: '气压(hPa)', min: 990, max: 1035},
  {key: 'precipitation', label: '降水量(mm)', min: 0, max: 10},
  {key: 'co2', label: 'CO₂浓度(ppm)', min: 405, max: 425},
  {key: 'humidity', label: '湿度(%)', min: 0, max: 100},
  {key: 'current_speed', label: '海流(m/s)', min: 0, max: 1}
];

// 计算坐标轴位置
const axisPositions = {};
dimensions.forEach((dim, i) => {
  axisPositions[dim.key] = (i / (dimensions.length - 1)) * width;
});

// 数据归一化
function normalizeValue(value, dimension) {
  return (value - dimension.min) / (dimension.max - dimension.min);
}

// 计算Y坐标(像素位置)
function getY(value, dimension) {
  const normalized = normalizeValue(value, dimension);
  return height - (normalized * (height - margin.top - margin.bottom)) - margin.bottom;
}
```

#### 3. 维度线渲染

**SVG路径生成**
```javascript
// 为每条数据记录生成路径
const polylines = svg.selectAll('.polyline')
  .data(data)
  .enter()
  .append('path')
  .attr('class', 'polyline')
  .attr('d', d => {
    // 构造SVG path: M x0 y0 L x1 y1 L x2 y2 ...
    let path = '';
    dimensions.forEach((dim, i) => {
      const x = axisPositions[dim.key];
      const y = getY(d[dim.key], dim);
      path += (i === 0 ? 'M' : 'L') + ` ${x} ${y}`;
    });
    return path;
  })
  .attr('stroke', d => colorScale(d.region))
  .attr('stroke-width', 1)
  .attr('opacity', 0.5)
  .attr('fill', 'none');

// 添加悬浮交互
polylines.on('mouseover', function(event, d) {
  d3.select(this)
    .attr('stroke-width', 2)
    .attr('opacity', 1);
  
  showTooltip(event, d);
})
.on('mouseout', function() {
  d3.select(this)
    .attr('stroke-width', 1)
    .attr('opacity', 0.5);
  
  hideTooltip();
});
```

**采样模式切换**
```javascript
// 采样函数
function sampleData(data, sampleSize) {
  if (data.length <= sampleSize) return data;
  
  const step = Math.floor(data.length / sampleSize);
  const sampled = [];
  for (let i = 0; i < data.length; i += step) {
    sampled.push(data[i]);
  }
  return sampled;
}

// 采样切换按钮
d3.select('#sample-mode-btn').on('click', function() {
  isSamplingMode = !isSamplingMode;
  const displayData = isSamplingMode ? sampleData(data, 900) : data;
  redrawPolylines(displayData);
  
  d3.select(this).text(isSamplingMode ? '采样模式(899条)' : '完整模式(3600条)');
});
```

#### 6. 性能优化

**优化策略**

1. **数据聚合**: 3600条→采样900条

- 蓝黄色盲(Tritanopia): 无法分辨蓝黄(罕见)

**配色方案对比**

#### 3. 格式塔原则优化

**格式塔原则应用**

1. **接近性(Proximity)**
   - 调整图表元素间距
   - 相关联的元素靠近放置
   - 应用示例: 图例与对应数据靠近

2. **相似性(Similarity)**
   - 同一类型的元素使用相同样式
   - 应用示例: 所有海域使用对应颜色
   - 结果: 视觉上快速识别数据类别

3. **连续性(Continuity)**
   - 保持视觉流向的连贯性
   - 应用示例: 桑基图的流向线条
   - 结果: 用户易于追踪数据路径
