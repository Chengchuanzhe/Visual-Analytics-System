## 二、数据来源

| 变量类别 | 来源 | 获取方式 |
|----------|------|----------|
| 海表温度、盐度 | IFREMER Argo 浮标观测数据（ArgoFloats） | ERDDAP REST API（无需注册） |
| 其余大气变量 | 基于气候学规律的合理估算 | Python 脚本生成 |

**Argo 数据地址**：`https://erddap.ifremer.fr/erddap/tabledap/ArgoFloats`

Argo 浮标是国际海洋观测计划部署的自动剖面浮标网络，覆盖全球主要海域，提供实测海表温度（temp）和盐度（psal）数据，数据质量经过国际标准化质控。

## 四、数据处理过程

### 步骤 1：数据采集
通过 ERDDAP REST API 按海域分别请求 Argo 浮标表层数据（压强 pres ≤ 15 dbar，等效于海表 15m 以内），时间范围 2020-01-01 至 2022-12-31。

```
GET https://erddap.ifremer.fr/erddap/tabledap/ArgoFloats.json
    ?time,latitude,longitude,pres,temp,psal
    &time>=2020-01-01T00:00:00Z
    &time<=2022-12-31T00:00:00Z
    &pres<=15
    &latitude>=LAT_MIN&latitude<=LAT_MAX
    &longitude>=LON_MIN&longitude<=LON_MAX
```

### 步骤 2：清洗与采样
- 过滤掉时间、经纬度、温度任意字段为 null 的记录
- 每海域随机采样 600 条（原始记录量：太平洋 195185、大西洋 450296、印度洋 256541、北极 40941、南极 14398、赤道 127885）

### 步骤 3：大气变量填充
基于纬度和季节的气候学模型估算风速、气压、降水等 9 个大气变量，保证物理合理性（如极地风速偏大、赤道降水偏高等规律）。

### 步骤 4：格式输出
合并为统一 JSON 数组，经度统一转换为 -180~180 范围，所有浮点数保留 2~4 位小数。

## 五、数据质量说明

| 质量项 | 状态 |
|--------|------|
| 海表温度 / 盐度 | 来自 Argo 国际标准质控数据（Quality Flag = 2，"probably good"） |
| 缺失值 | 已过滤，最终数据集无 null 的核心字段 |
| 大气变量 | 气候学估算，保证物理量级合理，不代表精确实测值 |
| 时间分布 | 2020/2021/2022 三年均有覆盖 |
| 地域分布 | 6 个海域各 600 条，分布均衡 |
