2. 格式塔原则优化明细

原则定义应用图表优化前问题优化措施
接近性 空间上靠近的元素被知觉为一组 饼图 图例距 SVG 32px 过大 缩至 20px
相似性 颜色/形状相似的元素被归为一类 雷达图 6 条同色系多边形不易区分 默认仅3条+图例切换
连续性 沿平滑路径排列的元素被知觉为连续 平行坐标图 折线过多(720条)形成色块 默认采样模式
闭合性 闭合形状被视为完整图形 雷达图 多边形首尾已闭合 ✓ 无需优化
主体/背景 主体从背景中分离感知 散点图/气泡图 印度洋黄点与白色背景融合 加 stroke:#8B8000
共同命运 同步运动的元素被视为一组 全部图表 hover 动画时长不一致 统一 0.2s ease

格式塔原则应用示例

接近性 (Proximity)

距离近=同组

相似性 (Similarity)

颜色相同=同组

连续性 (Continuity)

折线连续=趋势
