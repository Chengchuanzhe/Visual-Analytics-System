4. 视觉规范 v3.0 修订摘要

修订 1：黄色强制描边

所有使用印度洋黄色 #F0E442 的元素， 统一附加 1.5px #8B8000 描边。此规则写入党纪。

修订 2：Tooltip 统一

全部图表的 tooltip 使用同一套 CSS： rgba(20,40,70,0.92) 底 + #fff 13px 字 + 8px 圆角。

修订 3：动画曲线统一

所有交互过渡统一使用 transition: 0.2s ease（悬浮高亮/调暗/显隐）。

修订 4：卡片视觉统一

所有图表页面统一白底卡片 + 12px 圆角 + 0.06opacity 阴影 + 32px padding。

修订 5：印度洋双重编码

雷达图和散点图中印度洋 (#F0E442) 数据点附加交叉影线或虚线，作为形状双重编码。

修订 6：对比度底线

新增硬性规则：图表中任何"文字/数据标记" 与背景对比度不得低于 3.0:1。
