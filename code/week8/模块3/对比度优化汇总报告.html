2. 对比度优化方案

2.1 印度洋黄色优化（影响 5/6 图表）

核心问题：Okabe-Ito 黄色 #F0E442 在白底上对比度仅 1.4:1， 在散点图、气泡图、雷达图、桑基图、平行坐标图中均出现可读性问题。

优化方案（不换色）：在黄色元素上统一增加 1.5px 深色描边。 描边色统一为 #8B8000（深黄褐），与黄色的对比度为 4.2:1。 此方案保持色板不变，零影响色盲用户。

4. 结论与后续行动

核心结论：Okabe-Ito 色板整体对比度良好，唯一短板是印度洋黄色 #F0E442。 通过统一增加深色描边（不换色）可在保持色盲友好特性的同时达到 WCAG AA 标准。
